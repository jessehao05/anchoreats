### PWA Setup

A PWA (progressive web app) allows this website to be added to your home screen and behave just like an app. To setup this website to act like an app, follow the instructions below:

iPhone / iPad
- Open AnchorEats in Safari
- Tap the Share button
- Choose Add to Home Screen
- Tap Add

Android
- Open AnchorEats in Chrome
- Tap Install app if prompted, or open the browser menu
- Choose App to Home Screen or Install app
- Confirm Install

The app will be added to the device's home screen. This can be undone at any time by selecting the app on the home screen and selecting Delete Bookmark.
